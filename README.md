# Titanic Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python, Pandas, Matplotlib, and Seaborn.

## Objective
To identify patterns, trends, and relationships affecting passenger survival.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Titanic Dataset (train.csv)

## Analysis Performed
- Data Overview
- Missing Value Analysis
- Statistical Summary
- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis

## Visualizations
- Histogram of Age
- Boxplot of Fare
- Survival by Gender
- Survival by Passenger Class
- Correlation Heatmap
- Pairplot

## Key Findings
- Females had higher survival rates.
- First-class passengers survived more frequently.
- Fare and Passenger Class influenced survival.
- Missing values were found in Age, Cabin, and Embarked columns.
